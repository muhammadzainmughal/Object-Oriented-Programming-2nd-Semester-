# OOP Lab 1 - Structures in C++

## Overview
This lab focuses on the basic concepts of structures in C++. It includes the use of structures, arrays of structures, and structure pointers to store and manage data efficiently.

## Topics Covered
- Structures  
- Arrays of Structures  
- Structure Pointers  

## Tasks
- Create a Student structure with data members and a display function  
- Use arrays of structures to handle multiple students  
- Implement structure pointers for dynamic data access
