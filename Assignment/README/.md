# Airline Reservation & Flight Management System

## Description

This project is a simple Airline Reservation and Flight Management System developed in C++ using Object-Oriented Programming concepts.

## Features

- Add Flights
- Display Flights
- Register Passengers
- Display Passengers
- Book Tickets
- Cancel Tickets
- Exception Handling
- Inheritance
- Polymorphism
- STL Vector Usage

## Classes Used

- Flight
- DomesticFlight
- Passenger
- EconomyPassenger
- Ticket
- Airline

## OOP Concepts

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction

## How to Compile

```bash
g++ *.cpp -o airline
