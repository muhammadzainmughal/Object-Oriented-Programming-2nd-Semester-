# Project Description

The Vehicle Rental System is a menu-driven C++ application developed using Object-Oriented Programming concepts. It allows users to manage vehicles, register customers, rent vehicles, return rented vehicles, and generate a summary report.

The project demonstrates the use of inheritance, polymorphism, abstraction, encapsulation, and dynamic memory allocation.

---

# Features

- Add Vehicles
- Register Customers
- Rent Vehicles
- Return Vehicles
- Summary Report
- Cost Calculation
- Vehicle Availability Management

---

# Vehicle Types

### Car
- Stores number of seats.
- Rental cost is calculated using the daily rental rate.

### Motorbike
- Stores engine capacity (CC).
- Provides a 10% discount for rentals longer than 7 days.

### Truck
- Stores payload capacity.
- Adds a 20% surcharge to the total rental cost.
